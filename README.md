# volo_gif


# Run #
To run the project, follow the commands below,
1. clone the repository from  https://github.com/BanidAzin/volo_gif.git
2. cd volo_gif
3. npm install
4. For android
    1. npx react-native run-android
5. For iOS
    1. cd ios && pod install && cd ../
    2. npx react-native run-ios
